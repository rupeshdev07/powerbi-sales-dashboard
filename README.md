# 📊 Power BI Sales Dashboard

## 📌 Project Overview
This project is an interactive Sales Dashboard built using Microsoft Power BI. It helps analyze sales performance, targets, and branch-wise business insights through interactive visualizations.

## ✨ Features
- Sales Performance Analysis
- Branch-wise Sales Report
- Daily Sales Target Tracking
- Interactive Charts and Filters
- KPI Dashboard

## 🛠️ Tools Used
- Microsoft Power BI
- Microsoft Excel

## 📂 Files Included
- Pizza Sales Analysis.pbix
- Sales-Data.xlsx
- Sales-Target.xlsx
- Daily-Sales-Target.xlsx
- Branch-Data.xlsx

## 📈 Dashboard Insights
- Total Sales
- Sales by Branch
- Sales Target vs Achievement
- Daily Sales Performance
- Business KPIs

## 🚀 How to Use
1. Download the repository.
2. Open `Pizza Sales Analysis.pbix` in Power BI Desktop.
3. Refresh the data if required.
